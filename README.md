### 1.Add Two Number
注意审题:
2->4->3 代表逆序存储的342

5->6->4 代表逆序存储的465

7->0->8 代表逆序存储的807

所以,其实题目的意思是求两个这样的链表的和，并通用以链表方式表示出来。

解题思路:
只需要构建链表，存储一个进位即可(比如2+8,那么下一位的进位为1)。
注意最后进位如果不为0，需要增加一位(比如5 +5 结果为0->1)
