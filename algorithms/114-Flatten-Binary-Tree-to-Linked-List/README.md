## Flatten-Binary-Tree-to-Linked-List

## 说明
给出一颗二叉树，将它转成单链表(可以以前序遍历方式)，例如:

```
         1
        / \
       2   5
      / \   \
     3   4   6
```
转换成

```
   1
    \
     2
      \
       3
        \
         4
          \
           5
            \
             6
```

### 思路

* 1.先中序遍历，然后改变
* 2.先序迭代递归
* 3.非迭代法
