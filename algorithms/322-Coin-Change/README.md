## Coin-Change

## 说明

给你一些不同的硬币面额以及钱的总数，计算最少需要多少硬币可以生成这么多钱。

如果这些面额不能组成这个数的钱，返回`-1`（假设有无限多的硬币）

示例

```js
coins = [1, 2, 5], amount = 11
return 3 (11 = 5 + 5 + 1)
```

```js
coins = [2], amount = 3
return -1.
```

### 思路

- 动态规范

- `dp`表示钱数为i时的最小硬币数的找零

- 递推式为：`dp[i] = min(dp[i], dp[i - coins[j]] + 1);`

- 其中`coins[j]`为第j个硬币，而`i - coins[j]`为钱数i减去其中一个硬币的值，剩余的钱数在dp数组中找到值
