## Additive-Number

## 说明

加法数（`Additive Number`）的定义如下

- 至少包含`3`个数

- 除了第`1``2`个数，所有数字都必须正好等于前两个数的和

- 数字不能以`0`开头，`1, 2, 03`是非法的

例如

`112358` 是加法数，因为`1 + 1 = 2, 1 + 2 = 3, 2 + 3 = 5, 3 + 5 = 8`

`199100199`也是加法数，`1 + 99 = 100, 99 + 100 = 199`

给出一个字符串，字符范围时`0-9`，判断它是否是加法数

注意

- 接收的可能是大数
