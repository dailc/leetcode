# String-To-Integer

## 说明

实现`atoi`，将字符串转为整数

提示：

- 请考虑到所有可能的输入

注意

- 在第一个有效字符之前的所有空格都需要被舍弃

- 先可能会有正负号，接下来才是数字

- 除了完整数字外的字符都无效(譬如`-2ssss`后面的`sss`无效)

- 如果非空格字符不是有效数字，不进行转换（譬如`-s2`无效）

- 如果没有合法的转发，返回`0`

- 不能溢出`INT_MAX (2147483647)`,`INT_MIN (-2147483648) `

## 思路



示例中给出了几种基本解法

关键是要读懂题目，千万别读错了


比如:
-22knjn  结果为-22
-+22上述  为0
