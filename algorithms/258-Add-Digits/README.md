## Add-Digits

## 说明

给一个非负整数`num`，一直循环的将这个数的所有位数相加，得到新的数字，直到最后的数字是个位数，例如

```js
num = 38;
```

过程

```js
3 + 8 = 11;
1 + 1 = 2;
```

由于`2`是个位数，所以返回了它

注意

- 不能用循环或者递归

- 在`O(1)`的时间复杂度

### 思路

- 看规律，从`1，...，9,10，...,18,19,20`可以看出，结果分别是`1,...,9,1,...,9,1,2`

    - 每`9`个数一个循环
    
    - 因此`(num - 1) % 9 + 1`
