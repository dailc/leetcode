## Different-Ways-to-Add-Parentheses

## 说明

给出一个由数字和操作符组成的字符串，返回所有的可能计算结果，操作符只有`+`，`-`，`*`有效，例如

输入`2-1-1`，结果为`[0, 2]`

```js
((2-1)-1) = 0
(2-(1-1)) = 2
```

输入`2*3-4*5`，结果为`[-34, -14, -10, -10, 10]`

```js
(2*(3-(4*5))) = -34
((2*3)-(4*5)) = -14
((2*(3-4))*5) = -10
(2*((3-4)*5)) = -10
(((2*3)-4)*5) = 10
```

### 思路

- 分治法，递归，划分成子问题

    - 采用map进行优化
