## Lowest-Common-Ancestor-of-a-Binary-Search-Tree

## 说明
给出一颗二叉搜索树，以及两个节点，找到两个节点最近的共同祖先节点(一个节点可以是它自己的祖先节点)，例如

```
        _______6______
       /              \
    ___2__          ___8__
   /      \        /      \
   0      _4       7       9
         /  \
         3   5
```

`2`,`8`最近的共同祖先节点是`6`
`2`, `4`的最近共同祖先节点是`2`

### 思路

- 如果如果p,q 比root小, 则LCA必定在左子树, 如果p,q比root大, 则LCA必定在右子树. 如果一大一小, 则root即为LCA.
