# Word-Search

## 说明

给一个`2d`字符数组和一个单词，判断这个字符是否在这个数组中。这个单词可以是一个字符串。

其实就是找路径，例如

```js
[
  ['A','B','C','E'],
  ['S','F','C','S'],
  ['A','D','E','E']
]
```

word = "ABCCED", -> returns true,

word = "SEE", -> returns true,

word = "ABCB", -> returns false.

其中，数组中的每一个元素最多只能被使用一次，每一次只能横向或纵向走。

- 1.第一想法就是回溯法(考虑每一个元素作为起点的情况,然后每次都是从上下左右四周寻找)
